## 🚀 Projects

### 🌿 [CropGuard AI](https://github.com/ajinesh703/CropGuard-AI)
> **Crop disease detection system achieving ~96% accuracy**

An end-to-end AI application that helps farmers identify crop diseases early from leaf images. The backend uses a fine-tuned **MobileNetV2** model served via **FastAPI**, while the frontend is built in **React** for a responsive, user-friendly experience.

- 📦 **Stack:** FastAPI · React · MobileNetV2 · TensorFlow
- 🎯 **Highlights:** Transfer learning on plant disease dataset, REST API integration, real-time inference
- 🌾 **Impact:** Targets agricultural productivity in Indian farming communities

---

### ⚖️ Indian Legal RAG Chatbot
> **IPC-based legal assistant for accessible justice**

A Retrieval-Augmented Generation chatbot trained on the **Indian Penal Code**. Users can ask natural language questions and get contextually grounded answers backed by actual legal text, reducing the need for expensive legal consultations.

- 📦 **Stack:** LangChain · ChromaDB · Streamlit · Sentence Transformers
- 🎯 **Highlights:** Document chunking, semantic search over legal corpus, context-aware response generation
- 🏛️ **Use case:** Legal literacy tool for citizens unfamiliar with IPC provisions

---

### 🤖 Girlfriend Chatbot (RAG-based)
> **Personality-driven conversational AI with a sarcasm slider**

A creative RAG chatbot with adjustable personality traits — including a literal **sarcasm slider** that modulates tone and conversational style. Demonstrates how system prompts and retrieval pipelines can be combined to produce dynamic, consistent personas.

- 📦 **Stack:** FAISS · Sentence Transformers · Ollama · Streamlit
- 🎯 **Highlights:** Local LLM inference via Ollama, custom retrieval pipeline, personality parameterization
- 🧠 **Learnings:** Prompt engineering, personality encoding, local-first AI deployment

---

### 📄 ATS Resume Checker
> **Resume scoring engine using TF-IDF cosine similarity**

Helps job seekers optimize their resumes against job descriptions using NLP-based similarity scoring. Mimics how Applicant Tracking Systems evaluate candidates, giving actionable keyword-level feedback.

- 📦 **Stack:** Python · Streamlit · scikit-learn · NLTK
- 🎯 **Highlights:** TF-IDF vectorization, cosine similarity scoring, keyword gap analysis
- 📊 **Output:** A score out of 100 with missing keyword suggestions

---

### 🍎 Fruit Shelf Life Detector
> **Multi-output vision model for freshness classification and regression**

A computer vision model that simultaneously classifies fruit type and predicts days remaining before spoilage — a true multi-output learning problem combining classification and regression heads on **MobileNetV2**.

- 📦 **Stack:** MobileNetV2 · TensorFlow · Keras
- 🎯 **Highlights:** Custom multi-output model architecture, shared feature extractor, dual loss functions
- 🛒 **Application:** Smart inventory management for retail and home use
